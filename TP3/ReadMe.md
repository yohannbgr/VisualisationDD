# TP3 - Visualisation de Données Massives

## Description

Analyse de l'activité annuelle des 5 salons de beauté de Madame Lovelie (mars 2025 – mars 2026) à partir d'un dataset de 5 500 rendez-vous. L'objectif est de répondre à 9 questions analytiques couvrant le chiffre d'affaires, la saisonnalité, les annulations, la satisfaction client et les comportements de réservation, afin d'accompagner la prise de décisions stratégiques.
