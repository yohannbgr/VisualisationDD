# Visualisation de Données

Travaux pratiques et projet du cours de Visualisation de Données.

## Contenu
-TP2
-TP3
-TP4
-Projet d'analyse d'une base fictive de 44 991 emprunteurs issu d'un dataset Kaggle. Le projet couvre le nettoyage et l'enrichissement de la base en Python, puis la création de cinq tableaux de bord Tableau Public pour explorer le profil des emprunteurs, le risque de crédit, les caractéristiques des prêts, la performance et la segmentation financière.
## Technologies
- Tableau Public
- Python (pandas, matplotlib)
