# TP2 - Visualisation de Données

## Description

Ce TP a pour objectif d’analyser et de visualiser des données liées aux catastrophes naturelles à l’échelle mondiale.
L’étude porte sur différents types de catastrophes (séismes, inondations, tempêtes, incendies, etc.) afin d’identifier des tendances, des zones géographiques à risque et l’évolution de ces événements dans le temps.

Les visualisations permettent de mieux comprendre :

* la fréquence des catastrophes par type
* leur répartition géographique
* leur impact potentiel (nombre de victimes, dégâts, etc.)

## Données

Les données utilisées proviennent de bases ouvertes sur les catastrophes naturelles (ex : EM-DAT ou datasets Kaggle).
Elles contiennent des informations telles que :

* le type de catastrophe
* la localisation (pays, région)
* la date
* les impacts humains et économiques

## Objectifs du TP

* Nettoyer et préparer les données
* Créer des visualisations pertinentes

## Résultats principaux

* Les inondations et tempêtes sont les catastrophes les plus fréquentes
* La région la plus exposée est l'Asie
* Une augmentation globale du nombre de catastrophes est observable sur les dernières décennies
