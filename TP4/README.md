# TP4 - Visualisation de Données Massives

## Description

Analyse des résultats électoraux de 108 communes françaises à travers 9 visualisations interactives : répartition des communes par parti, participation régionale, distribution des voix par position politique, scores moyens par formation, communes abstentionnistes, heatmaps de performance territoriale et carte de France interactive.
